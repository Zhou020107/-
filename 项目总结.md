# 就业查询工具 — 项目总结

## 项目概述

跨平台就业信息爬取工具，自动采集招聘网站的职位数据，支持分类筛选、图表分析和 Excel 导出。

- **桌面端**：Streamlit 网页应用，Windows 运行，手机浏览器也可访问
- **手机端**：Flet 原生 APK，仅用 NCSS（国家大学生就业服务平台）

---

## 当前功能

### 数据采集
| 网站 | 方式 | 说明 |
|---|---|---|
| 51job（前程无忧） | Playwright 浏览器 | 新版 we.51job.com，支持按关键词搜索 |
| 国家大学生就业服务平台 | httpx JSON API | 无需浏览器，速度快，专为应届生 |

### 自动分类
| 分类项 | 说明 |
|---|---|
| 学历要求 | 博士/硕士/本科/大专/高中/不限 |
| 薪资范围 | <5K / 5K-10K / 10K-20K / 20K-35K / 35K+ |
| 工作地点 | 一线/新一线/二线/其他/远程 |
| 技术栈 | 350+ 关键词，覆盖IT、金融、建筑、医疗、教育、设计、制造、法律、运营、物流等 10+ 行业 |
| 招聘应届生 | 自动识别"应届生/校招/管培生/经验不限"等 |
| 五险一金 | 自动识别"五险一金/六险一金/社保/公积金"等 |
| 双休 | 自动识别"双休/周末双休/做五休二"等 |

### 数据图表（6种）
1. 学历要求分布
2. 薪资范围分布
3. 城市分布 TOP15
4. 技术栈需求分布
5. 来源网站分布
6. 招聘公司 TOP20

### Excel 导出
- 全部职位（主表，带格式/筛选/超链接）
- 按学历/薪资/城市/技术栈分类子表
- 统计摘要

### 浏览结果
- 筛选器：学历/薪资/城市/来源/应届生/五险一金/双休/全文搜索
- 排序：薪资↓↑ / 日期 / 公司 / 学历 / 地点
- 职位详情查看

## 启动方式

| 方式 | 操作 |
|---|---|
| 桌面端 | 双击 `启动.vbs` → 自动打开浏览器 |
| 手机访问桌面端 | 同一 WiFi，浏览器打开侧边栏显示的 IP |
| 手机独立 App | 下载 GitHub Actions 构建的 APK 安装 |

## 配置说明

编辑 `config.yaml` 可自定义：

```yaml
# 爬取速度（秒）
scraping:
  min_delay_seconds: 1
  max_delay_seconds: 3

# 自定义技术栈关键词
tech_keywords:
  custom:
    新能源: ["光伏", "风电", "储能"]
    农业: ["种植", "养殖", "育种"]
```
